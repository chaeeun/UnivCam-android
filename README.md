# 유니브캠(UnivCam) 📷

## 1. 개발 목적
스마트폰이 생기면서 손으로 노트필기하기보다 카메라어플을 이용해서 PPT, 칠판, 친구의 노트를 사진찍는 학생들이 많아졌습니다. 
하지만 스마트폰 갤러리안에 필기사진이랑 일반사진들이 섞여서 보기 불편합니다.
폴더정리가 있지만 사진찍은 후에 폴더정리를 따로 해줘야 해서 귀찮습니다.

## 2. 기대 효과
사진을 찍은 후에 폴더관리를 해야하는 번거로움을 덜어줍니다.
